# YouTube Sentiment Analysis API

Real-time sentiment analysis API for YouTube comments using Machine Learning.

## Endpoints

- `GET /health` - Check API health
- `POST /predict_batch` - Analyze sentiment of multiple comments

## Tech Stack

- FastAPI
- scikit-learn
- TF-IDF + Logistic Regression
- Docker
